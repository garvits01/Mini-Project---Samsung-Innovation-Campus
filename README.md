# 📊 India Health Indicators Analysis (NFHS-5)

## 📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on the National Family Health Survey (NFHS-5) dataset. The goal is to uncover hidden correlations between various health indicators across Indian states, specifically investigating the "double burden of malnutrition" (coexistence of undernutrition and obesity).

## 🛠️ Tools & Technologies Used
* **Python** (Data Manipulation & Analysis)
* **Pandas & NumPy** (Data Cleaning & Aggregation)
* **Matplotlib & Seaborn** (Data Visualization)
* **Folium** (Interactive Geospatial Mapping)

## 📂 Files in this Repository
* `NFHS5_Analysis.ipynb`: The Jupyter Notebook containing the full Python code, visualizations, and interactive maps.
* `datafile.csv`: The raw dataset containing state-wise health metrics.

## 📈 Key Visualizations Included
1. **Correlation Heatmap:** Analyzing the relationship between child anaemia and adult obesity.
2. **Bar Charts:** Highlighting the top 10 states with the highest rates of hypertension in men and women.
3. **Interactive Choropleth Map:** A geographic dashboard pinpointing public health risk zones across India.

## 🚀 How to Run
To view the interactive maps and charts, simply click on the `NFHS5_Analysis.ipynb` file above. Alternatively, you can download the repository and run it locally using Jupyter Notebook.
